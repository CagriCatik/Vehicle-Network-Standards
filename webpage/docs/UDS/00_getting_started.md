## 2. **Services**
Dieser Abschnitt beschreibt die verschiedenen Dienste und ihre Funktionalität im UDS-Protokoll.

- **Wichtige Inhalte:**
  - Diagnostic Session Control – 0x10
  - ECU Reset – 0x11
  - Security Access – Service 0x27
  - Read Data By Identifier – 0x22
  - Write Data By Identifier – 0x2E
  - Read Memory by Address – 0x23
  - Write Memory by Address – 0x3D
  - Tester Present Service – 0x03
  - Communication Control Service – 0x28
  - Read Scaling Data by Identifier – 0x24
  - Request File Transfer – 0x38
  - Request Download – 0x34
  - Request Upload – 0x35
  - Transfer Data – 0x36
  - Request Transfer Exit – 0x37
  - I/O Control – 0x2F
